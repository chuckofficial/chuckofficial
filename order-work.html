자동 발주

자동 발주를 통해 29CM, 서울스토어, 스타일쉐어, 무신사, 공식홈페이지의 발주 내역을 자동으로 불러올 수 있습니다.

이지오토 프로그램을 실행 후 로그인을 한 뒤에 ‘자동발주’ 버튼을 클릭해서 기다리시면 됩니다.

수동 발주

수동 발주는 자동 발주로 발주 내역을 자동으로 불러올 수 없는 거래처들의 발주 내역을 수동으로 파일로 불러와서 이지어드민을 통해 업로드를 시켜주는 작업입니다. 하지만 몇몇 거래처의 경우 이지어드민으로 업로드 하지않고 송장을 바로 생성해서 발송하기도 합니다.

이지어드민으로 발주내역 업로드하는 업체 목록 : SSF SHOP 신세계, 어라운드 더 코너, 에이랜드, LUUD(러드), 쿠팡

이지어드민으로 발주내역 업로드하지않는 업체 목록 : FDU(중국 해외배송 업체), KONVINI(일본 해외배송 업체), 카카오샵

@ 이지어드민으로 발주내역 업로드하는 업체

1. SSF SHOP 신세계

1. '주문/배송관리 - 주문현황'으로 접속합니다.

2. 검색을 누르신 후 나오는 모든 신규주문에 대해 체크하신 뒤 5번 항목인 '주문확인'을 누릅니다.

1. '주문/배송관리 - 배송현황'으로 접속합니다.

2. 검색을 누르신 후 3번 항먹인 '엑셀 다운로드'를 클릭합니다.

1. SSF 신세계의 경우 고객 전화번호에 국제 전화번호 표기가 되므로 +82를 삭제해줍니다.

1. '파일 - 다른 이름으로 저장' 클릭

1. 파일명을 '날짜 업체명'으로 변경합니다.

2. 파일 형식을 '쉼표로 구분된 값(.csv)'로 저장합니다.

2. 어라운드 더 코너

1. 사이트 로그인 후 주문관리 탭으로 접속합니다.

2. '주문관리 - 송장일괄등록' 메뉴로 접속합니다.

3. '엑셀다운로드' 버튼을 통해 발주된 목록을 다운로드 받습니다.

1. 파일 메뉴로 들어가서 다른 이름으로 저장을 클릭합니다.

1. 파일명을 다음과 같이 '날짜 업체명'으로 작성합니다.

2. 파일 형식을 쉼표로 구분된 값(.csv)로 저장합니다.

3. 에이랜드

1. 사이트 로그인 후 주문관리 탭에 마우스 커서를 올리면 하단 메뉴가 팝업됩니다.

2. '주문관리 - 입금확인리스트' 메뉴로 접속합니다.

3. 신규 주문을 확인하신 후 주문 전체를 체크합니다.

4. 그 다음 결제 상태를 '배송준비중'으로 변경하신 후에 저장을 누릅니다.

1. 다시 주문관리 팝업에서 '국내배송관리 - 배송준비중리스트' 메뉴로 접속합니다.

2. '엑셀양식선택'을 '양식'으로 변경 후에 엑셀로 저장을 통해 파일을 저장합니다.

1. 엑셀 파일을 실행하시고 '다른 이름으로 저장' 버튼을 누릅니다.

1. 파일의 제목을 '날짜 업체'로 변경하시고 파일 형식을 '쉼표로 구분된 값(.csv)' 파일로 저장합니다.

4. LUUD(러드)

1. '주문관리' 탭으로 들어가서 2번 항목인 '배송준비중 관리' 메뉴로 접속하시면 위와 같은 화면이 보여집니다.

2. 그 후 3번 항목인 '엑셀바로다운로드'를 눌러 파일을 다운로드 받습니다.

그 후 위와 같이 넘버스 파일로 실행이 된다면 모든 내용을 복사하셔야합니다.

정상적으로 엑셀 파일로 내용을 붙혀넣기 하신 후

'파일 - 다른 이름으로 저장'을 통해 파일의 제목을 '날짜 업체'로 변경하시고 파일 형식을 '쉼표로 구분된 값(.csv) 파일로 저장합니다.

5. 쿠팡

1. '주문/배송관리' - '배송관리'로 접속합니다.

2. 결제완료 탭에서 신규주문을 확인하고 주문건을 체크합니다.

3. '발주서 다운로드' 버튼을 통해 엑셀 파일을 다운로드 받습니다.

1. '발주서 다운로드' 버튼을 클릭하면 위와 같은 팝업창이 뜨는데 여기서 'CJ 대한통운'으로 택배사를 변경합니다.

엑셀 파일이 실행되면 주문 내역을 확인하신 후 '파일 - 다른이름으로 저장'으로 파일을 저장합니다.

1. '날짜 업체명'으로 제목을 변경합니다.

2. 파일 형식은 '쉼표로 구분된 값(.csv)'로 저장합니다.

@ 이지어드민으로 발주내역 업로드하지않는 업체

1. FDU(중국 해외배송 업체)

1. 주문관리로 들어갑니다.

2. '배송관리 - 송장입력목록'으로 접속합니다.

3. 주문 들어온 목록을 확인합니다(신규 주문의 경우 우측에 송장번호가 기입되어있지 않습니다.)

4. '송장엑셀파일다운로드'를 클릭해서 주문목록을 다운받습니다.

다운로드 된 파일의 경우 신규주문건만 따로 저장이 됩니다. 여기서 이지어드민을 통해

FDU 업체로 발송되는 송장을 생성하고 주문된 상품들을 송장에 넣습니다. (해당 엑셀은 출고 시 박스에 동봉하므로 프린터로 출력합니다)

위와 같이 송장을 생성한 뒤 똑같이 출고를 진행해주시면 되며 출력한 주문 내역 프린터물을 P03 비닐에 패킹 후 출고되는 박스에 동봉하시면 됩니다)

2. KONVINI(일본 해외배송 업체)

1. 로그인 후 주문관리로 들어갑니다

2. 출고 대기로 들어가시면 위와 같은 페이지가 출력됩니다.

3. 송장번호가 기입되지 않은 신규 주문을 체크하시고

4. '출고리스트다운로드'를 누르셔서 신규 주문 엑셀파일을 다운로드 받습니다.

엑셀 파일을 실행하신 다음 내용을 토대로 신규 송장을 만듭니다.(제품명, 컬러, 사이즈 정확히 확인)

3. 카카오샵

1. '카카오톡 스토어'로 접속합니다.

2. 관리자 인증 후에 '배송 요청'으로 들어가면 신규 주문 확인이 가능합니다.

3. 송장번호가 기입되지 않은 주문들이 신규주문이며 해당 신규주문의 송장을 직접 생성해서 송장을 출력하시면 됩니다.

수동 발주 파일 업로드

모든 수동 발주 파일들을 저장하신 후에 진행하게 되는 작업입니다.

이지어드민 사이트의 주문/배송관리 메뉴를 클릭하면 바로 보여지는 화면의 일부분을 캡쳐한 부분입니다.

1번 항목에서 판매처 이름을 확인하셔서 2번 항목의 업로드 버튼을 통해 저장한 파일을 저장합니다.

업로드 버튼을 누르면 뜨게되는 팝업입니다. '파일 선택' 버튼을 통해 정확한 파일을 업로드합니다.

업로드 진행이 완료되면 하단의 그래프에서 업로드한 업체의 그래프가 변동됩니다.

매칭 관련 설명

자동발주와 수동발주가 끝나고 난 후 이지어드민 페이지에서 매칭이 진행되며 기본적으로 주문배송관리 메뉴를 누르면 바로 연결되게 됩니다.

step1. 발주 단계에서는 발주가 된 내역이 확인되며 자동발주가 되지 않는 업체들의 파일을 업로드 할 수 있습니다.

step2. 메모 단계는 고객들의 메모를 확인 할 수 있는 화면이며 건너뛰어도 무관합니다.

step3. 매칭 단계에서는 매칭버튼을 누른 뒤 해당 주문에 맞는 상품을 정확하게 찾아서 매칭해주셔야 합니다.

step3-1. 1번 항목에서 상품명과 옵션을 확인하고, 2번 항목에 상품을 검색해서 오른쪽의 선택을 누르시고 3번 항목에서 맞는지 확인합니다.

step4. 합포 단계에서는 수령자와 주소, 전화번호가 동일하면 합포를 진행하며 다를 경우에는 상품을 따로 받기위해 주문하셨을 경우가 있으니 합포를 진행하지 않습니다.

폰케이스 발주 방법

폰케이스의 경우 자동발주와 수동발주를 끝마친 후 매칭까지 끝내시고 진행하셔야 됩니다.

이지어드민 내에서 '주문배송관리' 메뉴를 클릭한 후 '배송관리-주문다운로드' 메뉴로 접속합니다.

검색 전에 몇 가지의 옵션을 변경해주어야 합니다.

1. 전체 공급처를 'CHUCKCASE'로 변경합니다.

2. 사용자 정의를 '폰케이스'로 변경합니다.

3. 검색을 누르면 주문이 들어온 폰케이스를 확인하실 수 있습니다.

4. 발주된 상품을 확인합니다.

5. 파일생성 버튼을 클릭해 xlsx 확장자로 저장합니다.

2. 여기서 3번 항목인 '주문번호'를 클릭하시면

4. 러드

2. 3번 항목에서 택배사와 송장 번호를 기입하신 후 4번 항목 '배송대기 처리'를 클릭합니다.

1. 그 후 '주문관리 - 배송대기 관리' 메뉴로 들어가셔서 상품 체크하신 후 '배송중 처리'를 클릭하시면 완료됩니다.

5. 쿠팡

'주문/배송관리' 메뉴에서 배송관리로 접속합니다.

그리고 주문번호를 클릭하면 주문자정보와 수취인정보를 확인하실 수 있는데 성함을 C/S창에 검색합니다.

그 후 'CJ 대한통운' 택배사로 변경하신 후에 C/S창에서 확인한 운송장번호를 입력하고 '선택물품배송'을 클릭합니다.
